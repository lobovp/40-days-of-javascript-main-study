# 40 Days of JavaScript — Study

Репозиторий с учебными примерами и заданиями по курсу "40 Days of JavaScript".

Структура:

- `day-01/`, `day-02/`, ... — папки с заданиями по дням.

Как пользоваться:

- Откройте нужную папку и запустите HTML-файл в браузере (например, `index.html`).
- Для запуска скриптов можно открыть `script.js` и `test.js` в редакторе.

Git:

- Чтобы коммиты были подписаны именем и email, установите их глобально:

```
git config --global user.name "Ваше Имя"
git config --global user.email "you@example.com"
```

Или установить только в этом репозитории:

```
git config --local user.name "Ваше Имя"
git config --local user.email "you@example.com"
```
